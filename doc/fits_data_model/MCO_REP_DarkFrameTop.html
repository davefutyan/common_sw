MCO_REP_DarkFrameTop

MCO_REP_DarkFrameTop

Brief: Dark Frame of the top CCD margin area
Description: The Frame is a result of dark M and C observations. It can be used to update the REF_APP_DarkFrameTop The bias value is already subtracted and it is corrected for non-linearity.
