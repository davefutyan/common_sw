REF_APP_BiasOverscanRightFrame

Brief: Calibration Product: bias frame of the CCD margin area overscan right
Description: The image cube consist of 6 images. The standard data are as describe here. The REF_APP_BiasFrameMetadata defined the actual data in the FITS file. 1 = bias frame in ADU, to be used for read-out frequency 230 kHz 2 = bias error frame in ADU, to be used for read-out frequency 230 kHz 3 = RON (read-out noise) in ADU, to be used for read-out frequency 230 kHz 1 = bias frame in ADU, to be used for read-out frequency 100 kHz 2 = bias error frame in ADU, to be used for read-out frequency 100 kHz 3 = RON (read-out noise) in ADU, to be used for read-out frequency 100 kHz
