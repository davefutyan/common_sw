MPS_PRE_VisitConstraints

Brief: Table of visit constraints, created by the mission planning system
Description: There shall be one FITS file with such a table as second extension per short term planning cycle, i.e. one per week . The first extension is always a MPS_PRE_Visit data structure of the same planning cycle. The parameters of one visit constraint at a given time are defined in one row of the table. There shall be one set of values per minute. The rows shall be ordered by increasing time.
