REF_APP_WhiteCCDLocationPSFMetadata

Brief: Calibration Product : Meta data for the wavelength integrated PSFs, stored in the same FITS file
Description: There is one row per two dimensional image in the associated image cube. It defines the offset of the PSF image on the full CCD without margins. The offset is defined as the difference in pixels between the lower left pixel of the full CCD and the lower left pixel of the PSF image on the full CCD.
