SCI_PRW_FullArray

SCI_PRW_FullArray

Brief: L05 Product : raw full array image.
Description: There is no processing step applied. The pixel values are as they were received from the instrument. Data received during one pass are stored in this data structure. The image size may change if overscan pixels and dark regions are part of the image that was sent to ground.
