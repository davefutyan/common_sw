REF_APP_HkParamConversion

Brief: Defines the conversion curve that has to be applied for Hk parameters.
Description: Some HK parameters are defined as enum numbers. The conversion from the enum number to a meaningful text is defined the REF_APP_HkEnumConversion. This table defines the enum conversion by its CALIB_NAME that should be used for a specific Hk parameter.
