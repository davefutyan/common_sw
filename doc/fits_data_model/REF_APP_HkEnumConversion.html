REF_APP_HkEnumConversion

Brief: Conversion between enum numbers in HK TM data to text
Description: Some HK parameters are defined as enum numbers. This table shall be used to convert the enum number to a meaningful text. Each line defines a conversion from one enum number to its text for a specific HK parameter. There are always at least 2 rows, i. e. 2 conversions per HK parameter.
