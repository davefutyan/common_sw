REF_APP_DarkColumns

Brief: Defines the not corrupted dark columns of the CCD
Description: There is one row in this table. The value of column LEFT_DARK defines as a bit pattern the columns of the left dark margin which are not corrupted (corresponding bit = 1). Similar the value in RIGHT_DARK defines the not currupted columns of the right dark margin.
