REF_APP_EventParamConversion

Brief: Defines the conversion curve that has to be applied for event parameters.
Description: Most Event parameters are defined as enum numbers. The conversion from the enum number to a meaningful text is defined the REF_APP_EventEnumConversion. This table defines the enum conversion by its CALIB_NAME that should be used for a specific Event parameter.
