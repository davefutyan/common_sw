EXT_APP_SAAMap

Brief: Describing the SAA at a specific altitude.
Description: The purpose of this table is to define the SAA. Each row of the table defines for a point on a latitude / longitude - net if it inside the SAA (SAA_FLAG == true) or if it is outside the SAA (SAA_FLAG == false).
