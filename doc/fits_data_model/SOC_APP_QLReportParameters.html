SOC_APP_QLReportParameters

Brief: Defines the parameters of QL Reports.
Description: Defines the parameters by their name and data structure, where they are stored, that shall be provided in Quick Look reports. Each line define for one parameter in which QL report it shall be provided. A parameter may be provided in only one report or several. To be able to provide a parameter in the Long Term Trend Report the Aggregated column has to be set to True as well.
