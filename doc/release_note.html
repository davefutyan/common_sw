This corrects a bug where, when these objects were used as dict keys, or in sets, each object was considered unique even though their class attributes had the same value. Now, objects having the same values will be mapped to the same key, or set item. Because __hash__() shall only be implemented for immutable classes, the clear() method has been removed from the UTC class.
