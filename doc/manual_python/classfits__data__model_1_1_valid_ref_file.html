Detailed Description

This class is used to get the reference file that is applicable for a given UTC time.

Author
Anja Bekkelien UGE

The class takes list of reference file names in the constructor. The className argument specifies the data structure type of the files. The method getFile will return, among those files, the one that is applicable for a specific UTC time.
