Detailed Description

To store a UTC time.

Author
Reiner Rohlfs UGE

This class can store a UTC time, including fractions of seconds.

◆ __init__() [2/3]

def __init__ ( utc  )

Initializes the class.

param utc [string] The UTC time of this instance. The format has to be either yyyy-mm-ddThh:mm:ss.ffffff or TUyyyy-mm-ddThh-mm-ss. The second option is a sub-string of a filename. The fraction of the seconds, i.e. ffffff is optional. The dot (.) between ss and ffffff must not appear if ffffff is not defined.

Exceptions
runtime_errorif the utc string does not follow the required format.
