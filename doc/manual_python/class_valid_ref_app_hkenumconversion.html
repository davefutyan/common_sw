Detailed Description

This class is used to get the reference file of data structure type RefAppHkenumconversion that is applicable for a given UTC time.

Author
Anja Bekkelien, UGE The class takes a list of reference file names in the constructor. The method getFile will return, among those files, the one that is applicable for a specific UTC time.
