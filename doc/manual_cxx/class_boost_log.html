Configures logging to file.

Parameters
[in]procNameThe process name.
[in]procVersionThe process version.
[in]outDir(optional) The directory in which the output log file is created. If this parameter is not given, the file is created in the current directory
[in]fileName(optional) the name of the log file, without path. If this parameter is not provided, a unique file name containing the current UTC time is generated.
[in]revisionNumber(optional) The archive revision number
[in]processingNumber(optional) The processing number
[in]visitId(optional) the visit id as a string formatted as PRppnnnn-TGoooonn. If this argument is provided, the visit id is included in the file name.
[in]passId(optional) the pass id as a string formatted as PSyymmddhh. If this argument is provided, the pass id is included in the file name.
[in]obsid(optional) the obsid
[in]processingChain(optional) the processing chain. If provided, it is included in the log file name.
[in]logLevel(optional) the lowest log level that is written to file. Defaults to DEBUG.
Returns
the path to the log file
