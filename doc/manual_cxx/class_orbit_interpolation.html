Detailed Description

A class to perform interpolation in time for the data stored in AUX_RES_Orbit and MPS_PRE_VisitConstraints.

Author
David Futyan, UGE

An instance of this class can be used to perform interpolation of the data stored in AUX_RES_Orbit and MPS_PRE_VisitConstraints to a user specified time
