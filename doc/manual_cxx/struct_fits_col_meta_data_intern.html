Detailed Description

Structure to store information of existing columns in the open FITS table.

Author
Reiner Rohlfs UGE

This Structure is used only internally in FitsDalTable. This structure is used only in READONLY mode. It is filled while the FITS table is opened with meta data of all the columns of the table. It is used in the FitsDalTable::Assign function, if the user wants to set a variable to an already existing column.
