Detailed Description

Provides reading of a sequence of CCSDS formatted values in a memory buffer.

Author
Florian George ELSE

No memory allocation or copy are made by this class, it only provides access to CCSDS formatted values in a given memory buffer. Ownership of the memory block belong to the caller and must remain valid to read values.
