Returns a valid trigger file name.

Author
Anja Bekkelien UGE

If a processor name is provided, the format of the file name is:

path/CH_visitId_passId_utc_TRIGGER-processorName_V000.xml

Otherwise the processor name is left out of the file name:

path/CH_visitId_passId_utc_TRIGGER_V000.xml

Parameters
[in]paththe directory in which the output file will be saved.
[in]processorNamethe processor name as defined in the job order element "Processor_Name".
[in]validityStartthe validity start.
[in]visitIdthe visit id.
[in]passIdthe pass id.
[in]dataNamean additional identifier of the data
Returns
a file name created from the input arguments.
