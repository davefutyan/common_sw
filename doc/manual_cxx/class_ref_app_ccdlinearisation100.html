Detailed Description

The boundaries and the coefficients of the cubic spline for read-out frequency of 100 kHz.

Author
Reiner Rohlfs UGE

A spline function is used to correct for the non linearity of the CCD. The linearisation has to be applied on the e- values. The coefficients of a given row are valid from the e- value defined in the BOUNDARY column of that row up to the e- value (column BOUNDARY) defined in the next row. The e- of the last row is the highest number of electrons for which a correction can be applied.

Parameters
[in]namename of the keyword for which the value, the comment and the unit is requested
[out]commentthe comment, without the unit, is copied into this variable, if it is not a nullptr. comment is set to an empty string if no comment is available for the keyword.
[out]unitthe unit is copied into this variable, if it is not a nullptr. Note: the unit in a FITS keyword is placed in square brackets [] at the start of a comment string. unit is set to an empty string if no unit is available for the keyword.
Returns
the value of the keyword in the requested data type. A conversion from the ASCII text in the FITS header is performed if possible. The value can be requested in following data formats:
  • int8_t
  • uint8_t
  • int16_t
  • uint16_t
  • int32_t
  • uint32_t
  • int64_t
  • float
  • double
  • bool
  • std::string
Exceptions
runtime_errorif the keyword name does not exist
runtime_errorif the conversion to the requested data type cannot be performed
runtime_errorif the value in the FITS file is either too big or to small for the requested data type.
