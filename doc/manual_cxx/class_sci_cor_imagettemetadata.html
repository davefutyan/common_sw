Detailed Description

L1 product : Meta data of the corrected images, stored in the same FITS file.

Author
Reiner Rohlfs UGE

There is one row per two dimensional imagette in the associated image cube. It stores meta data of that imagette.

Returns the value of an attribute in the requested data type.

Parameters
[in]namename of the keyword for which the value, the comment and the unit is requested
[out]commentthe comment, without the unit, is copied into this variable, if it is not a nullptr. comment is set to an empty string if no comment is available for the keyword.
[out]unitthe unit is copied into this variable, if it is not a nullptr. Note: the unit in a FITS keyword is placed in square brackets [] at the start of a comment string. unit is set to an empty string if no unit is available for the keyword.
Returns
the value of the keyword in the requested data type. A conversion from the ASCII text in the FITS header is performed if possible. The value can be requested in following data formats:
  • int8_t
  • uint8_t
  • int16_t
  • uint16_t
  • int32_t
  • uint32_t
  • int64_t
  • float
  • double
  • bool
  • std::string
