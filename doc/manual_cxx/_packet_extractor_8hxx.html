using PacketCallback = std::function<void(CcsdsPacketReader packetReader, bool reconstructed)>

Callback function for each packet extracted from the telemetry frames stream.

Parameters
[in]packetReaderThe CCSDS Packet Reader of the extracted packet.
[in]reconstructedIndicates whether the packet was split across frames and had to be reconstructed. When true, the memory holding the packet is only valid during the callback and has to be copied to be used outside of the callback.
