Detailed Description

To read the leap seconds from a REF file and provide number of leap seconds for a specific time.

Author
Reiner Rohlfs, UGE

This class can read the number of leap seconds from the SOC_APP_LeapSeconds data structure and can return the number of leap seconds that have been introduced until a specific time.

Currently only the UTC class is using this class to retrieve the number of leap seconds.

◆ setLeapSecondsFileNames()

void LeapSeconds::setLeapSecondsFileNames ( const std::list< std::string > &  fileNames)
static

Reads the leap seconds from a SOC_APP_LeapSeconds data structure.

If in the list of fileNames are more than 1 leap second file the most up to data file at the time when this function is called is used.

This function has to be called for the first call of getNumLeapSeconds(). It is called by the CheopsInit() function.

The FITS extension, define by fileNames has to have two rows: UTC_TIME and TAI_UTC.
