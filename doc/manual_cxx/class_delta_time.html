Detailed Description

A class to store a delta time between two UTC times or two OBT times.

Author
Reiner Rohlfs, UGE

This class can be used to store a delta time between two times. It is the result of the "operator -" of the UTC class and of the "operator -" of the OBT class.
The "operator *" and the "operator /" can be used to modify the time difference and associate the time difference from a UTC time difference to a OBT time difference and vice versa.

usage: to calculate the UTC1 for a specific OBT1 while the correlation between UTC2 and OBT2 and the correlation slope is known:
UTC1 = UTC2 + (OBT1 - OBT2) * slope
