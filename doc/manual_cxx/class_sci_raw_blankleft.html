Detailed Description

Data of the blank CCD margin area on left side of the CCD.

Author
Reiner Rohlfs UGE

Depending on the value of MRG_PROC the data can be either the complete margin image (MRG_PROC = image), 3 values per row (MRG_PROC = row collapsed) or just 4 values in total (MRG_PROC = total collapsed) In reduced and total collapsed mode the header keywords MRG_DTYx define for each column in the image the type of data. It can be "mean", "stdev", "median" or "mad".
