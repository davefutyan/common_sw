Detailed Description

template<class HK_PRW_DAL_TABLE>
class HkPrwFitsTableChild< HK_PRW_DAL_TABLE >

Class to fill one or more PRW_HK FITS tables from one TM packet.

Author
Reiner Rohlfs UGE

The only purpose of this template class is to keep the fits_data_model data type of the table, to be able to fill some keywords. See destructor.

Template Parameters
HK_PRW_DAL_TABLEhas to be a SCI_PRW_HK* class
