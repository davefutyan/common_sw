Root class to create RAW HK Tables with either random values or application defined values.

There shall be only one instance per program of this class.

[in]outDirRAW HK files will be create in this directory.
[in]visitIdThis visit ID will be part of the filename and header keywords will be set with this visitID value.

const T &  value 
)
inline

Updates the value of one column.

At the next calls of writeRow this value will be written to the column until it is updated again by this function. The column will not be updated randomly any more after the value is set by this function at least once.

Parameters
[in]colNamename of the column for which the value will be set
[in]valuethis value will be set to the column.
Exceptions
runtime_errorif colName does not exist in any RAW HK table.

Writes one row to the table of structName.

The FITS file with the named table will be created if this function is called for the first time for a given structName.
