Detailed Description

L05 Product : Meta data of the unstacked images, stored in the same FITS file.

Author
Reiner Rohlfs UGE

There is one row per unstacked two dimensional image in the associated image cube. It stores meta data of that image.. This data structure is used for SubArrays as well as for images of the FullArray. In the later case there will be just one row in the table. The time is converted in UTC and MJD. The CE_COUNTER can be used to associate the unstacked image to a stacked image. All unstacked images that are stacked on board to one stacked image have the same CE_COUNTER.
