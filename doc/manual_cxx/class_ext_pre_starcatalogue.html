Detailed Description

CHEOPS Star Catalogue.

Author
Reiner Rohlfs UGE

One table list the target star and the background stars of one visit. It is derived from CHEOPS Star Catalogue. First the EXT_DRFT_StarCatalogue is created by the star_catalogueExtraction tool. visit_combination then creates the EXT_PRE_StarCatalogu and sets the VISITCTR.
