Detailed Description

A class to store an On-Board-Time (OBT)

Author
Reiner Rohlfs, UGE

A OBT is defined as seconds + fraction of seconds since an arbitrary start. The length of a OBT second may be slightly different than a UTC second. The least significant bit is used to define whether the OBT of the instrument was synchronized (1) with the spacecraft OBT or not (0).
As soon as the OBT is used in the CHEOPS system an OBT reset counter is assigned to the time. This reset counter is stored in the two most significant bytes.

Initializes the on-board time of the class.

The obt is defined as the SC Time in the TM packets: 1 time tick is 2 ** -32 seconds.

Parameters
[in]obtthe OBT time, including synchronization bit, but without reset counter
[in]utcThe approximately time of the obt as UTC. The UTC time does not need to be very precise. But should be in the range of +/- 2 days. It is used to identify the correct OBT reset counter. If it is not define (used the default UTC time) or if no ObtReset file was provided then the OBT reset counter is not updated.

The synchronized bit can also be set as bit 0 of the obt.
