Detailed Description

L1 product : Meta data of the corrected images, stored in the same FITS file.

Author
Reiner Rohlfs UGE

There is one row per two dimensional image in the associated image cube. It stores meta data of that image. This data structure is used for subArrays as well as for images of the FullArray. In the later case there will be just one row in the table.
