Detailed Description

Implementations of the FitsDalHeader class.

Author
Reiner Rohlfs UGE
Version
9.3.1 2018-06-25 RRO #16553 add the const specifier to GetFileName()
9.2 2018-05-15 ABE #15807 Set PIPE_VER in data product headers
6.2 2016-09-02 RRO #11463: set version of data in every header
5.2 2016-06-05 RRO new Method: WriteCurrentStatus()
4.4 2015-12-04 RRO : GetFileName() returns the filename without extension.
4.2 2015-09-07 RRO #9181: set PROC_CHN keyword
3.0 2014-12-17 RRO #7054: support uint64_t as datatype for a keyword value. It is needed to set TNULL for a uint64_t column.
1.3 2014-07-31 RRO #6101: Grouping of header keywords
1.2 2014-07-30 RRO #6099: the order of keywords in the FITS header is defined by the order in the fsd file.
1.2 2014-07-29 RRO #6096: new function: CompareDataStructureVersion()
1.1 2014-07-24 RRO #6076: write keywords DATASUM and CHECKSUM when a DHU is created / updated. New method: FitsDalHeader::VerifyChecksum()
1.0 2013-12-14 RRO first released version
