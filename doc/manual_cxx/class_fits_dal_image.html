Detailed Description

template<typename T>
class FitsDalImage< T >

Representing a FITS image with access methods to the data of the image.

Author
Reiner Rohlfs UGE

Following data types of the image (T) are supported:

  • uint8_t
  • int16_t
  • uint16_t
  • int32_t
  • uint32_t
  • int64_t
  • float
  • double

Opens / creates a FITS image on disk.

Depending on the mode it creates a new image or opens an existing FITS image.

Parameters
[in]filenamefilename of the FITS file with the new or existing image extension. If mode is READONLY it can include the specification of the FITS extension to be opened.
[in]modeREADONLY: opens an existing FITS image extension
CREATE: creates a new FITS file with a new image extension. Throws an exception if the file exist already.
APPEND it is identical as CREATE if the file does not yet exist. A new extension is appended in the file if it exist already.
[in]sizeThe size of each dimension. For example {2, 3, 4} will create a 3 dimensional image with a X-Axis - size of 2 pixels, a Y-Axis size of 3 pixels and a Z-Axis of 4 pixels.

Returns the NULL value of the image.

The return.first is only valid if return.second == true. Note: The NULL value of an image has to be defined in the Fits Schema Document (fsd) file. ◆ ResizeThirdDimension()

template<typename T >
void FitsDalImage< T >::ResizeThirdDimension ( long  size)

re-sizes the third dimension of the image to size

Any iterator on the image will be violated by this method! Only images opened in CREATE or in APPEND mode can be updated. This method should be used carefully. It is very processing time intensive as the FITS file is re-structured each time. All extension following this image will be shifted inside the file.

Parameters
[in]sizenew size of the third dimension.
Exceptions
runtime_error
  • if he image has not exactly 3 dimensions
  • if the image was opened in READONLY mode.
  • if the size is < 0
  • if there is not enough memory to copy the data from the previous data buffer to the re-sized data buffer
  • if the FITS file cannot be re-sized, i.e. if cfitsio returns an error
